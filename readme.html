Mapty

About the Project

Mapty is a modern application for tracking your workouts. It combines the power of geolocation, maps, and data storage to help users log their running and cycling activities. With an intuitive interface and interactive map features, users can see where they exercised and keep track of their workout performance.

Live Demo: https://tanzeel0.github.io/Mapty/

Key Features

  • Geolocation-enabled to track workout starting points.
  • Log running and cycling activities with specific details (distance, duration, and workout type).
  • Interactive map display to visualize your workouts on the map.
  • Persistent data storage: workouts are saved and loaded from the local storage.
  • Responsive design for optimal viewing on desktop and mobile devices.

How to Use

1. Visit the live demo link and allow location access.
2. Choose between running or cycling.
3. Fill in the distance and duration of the workout.
4. Click on the map to set the location of your workout.
5. Your workout will be logged and displayed on the map!
